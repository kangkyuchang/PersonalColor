# 프로젝트 개요
### 해결하려는 문제 정의
기존의 퍼스널 컬러 진단 방식은 전문 컨설턴트의 **주관적인 판단**과 **시간/비용**을 필요로 합니다.
또한, 진단을 위해서는 **살제 드레이프(천)를 얼굴에 직접 대보는 과정이**이 필수적이므로,
사용자는 시간과 장소의 제약을 받으며 진단 결과의 **객관성**과 **일관성**을 확보하기 어렵습니다.

### AI/딥러닝 선정 이유
퍼스널 컬러 진단은 얼굴의 **미묘한 톤, 명도, 채도, 대비** 등 복잡한 시각적 특징을 **정밀하게 분석**하고 **색상 팔레트와 매칭**해야 하는 고차원적인 작업입니다.

일반적인 프로그래밍 로직으로는 이러한 **미묘한 시각 패턴과 피부 톤의 변화를 정량적으로 인식하고 분류**하는 것이 불가능합니다. 
따라서 **딥러닝 기반의 이미지 인식 및 분석 기술**을 활용하여:
1. 사용자의 **사진 한 장**만으로 얼굴 톤 영역을 정확하게 식별하고,
2. 전문가의 진단 결과를 **객관적인 데이터**로 학습하여,
   **정확도가 높은 자동화된 퍼스널 컬러 진단 시스템**을 구축하고자 했습니다.

## 최종 목표 및 평가 지표
본 프로젝트는 전문적인 컨설팅 없이도 신뢰할 수 있는 결과를 제공하는 **퍼스널 컬러 진단 AI 모델 개발**을 목표로 합니다.
| 목표 항목 | 핵심 내용 |
|:---|:----|
| **문제 해결 목표** | **사용자의 사진 한 장**만으로 퍼스너 컬러(예: 봄 웜톤, 겨울 쿨돈 등)를 진단 |
| **성능 목표** | 전문가의 진단결고와 비교하여 **높은 정확도** 달성 |
| **핵심가치** | **언제 어디서든** 제약 없이 자신의 퍼스널 컬러를 **객관적이고 일관된 결과**로 확인 가능한 서비스 제공 |
